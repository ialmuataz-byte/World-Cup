# 🏆 تحدي المونديال

**مسابقة كروية بين لاعب ولاعب أو فريق وفريق — من يملك الذاكرة الأقوى؟** ⚽

صفحة ويب مستقلة بالكامل (HTML + CSS + JavaScript في ملف واحد) لمسابقة أسئلة في عالم كرة القدم، مع تركيز على كأس العالم عبر نسخه المختلفة، واللاعبين، والمنتخبات، والأندية، والأرقام القياسية، واللحظات التاريخية.

## ✨ المميزات

- ⚔️ مواجهة بين طرفين: لاعب ضد لاعب، فريق ضد فريق، أو مجموعة ضد مجموعة
- 🎲 46 سؤالًا في بنك الأسئلة، تُختار عشوائيًا بدون تكرار (10 أسئلة لكل طرف بالتناوب)
- 📊 لوحة نتيجة حية تُبرز الطرف المتقدم
- 🌟 سؤال ذهبي لحسم التعادل
- 🎉 تأثير احتفالي (كونفيتي) عند إعلان الفائز
- 💾 حفظ آخر نتيجة في localStorage
- 📲 مشاركة النتيجة عبر WhatsApp
- 📱 تصميم عربي RTL متجاوب للموبايل والكمبيوتر، بخط Tajawal وألوان أخضر وذهبي

## 🚀 النشر على GitHub Pages

1. أنشئ مستودعًا جديدًا على GitHub (مثلًا: `football-quiz`).
2. ارفع جميع ملفات هذا المجلد إلى جذر المستودع.
3. من صفحة المستودع افتح: **Settings ← Pages**.
4. تحت **Source** اختر: **Deploy from a branch**، ثم الفرع **main** والمجلد **/ (root)**، واضغط **Save**.
5. انتظر دقيقة تقريبًا، وسيكون موقعك متاحًا على:

```
https://اسم-حسابك.github.io/football-quiz/
```

## 🕹️ طريقة اللعب

1. أدخل اسم الطرفين واختر نوع المواجهة، ثم اضغط **ابدأ التحدي**.
2. تظهر الأسئلة بالتناوب بين الطرفين (10 أسئلة لكل طرف).
3. اقرأ السؤال على الطرف الذي عليه الدور، ثم اضغط **إظهار الإجابة** واحكم: **إجابة صحيحة ✅** أو **إجابة خاطئة ❌**.
4. كل إجابة صحيحة = نقطة واحدة.
5. في النهاية يظهر الفائز — وإذا تعادل الطرفان، يُحسم اللقب بـ **السؤال الذهبي** 🌟.

## ✏️ إضافة أسئلة جديدة

افتح `index.html` وابحث عن المصفوفة `QUESTIONS` داخل وسم `<script>`، ثم أضف سؤالك بهذه الصيغة:

```js
{
  question: "نص السؤال",
  answer: "الإجابة الصحيحة",
  category: "كأس العالم",
  difficulty: "سهل"
}
```

> لا تنسَ تعديل `football-quiz.html` أيضًا إذا كنت تستخدم النسختين.
